# Craftable Nametag (Fabric)

Minecraft **1.20.1**, **Fabric Loader** ≥ **0.18.4**. Adds a **shaped** recipe for the vanilla name tag: one row, left to right — **iron ingot**, **leather**, **paper**.

## Build

```bash
./gradlew build
```

JAR: `build/libs/`.

## Develop

```bash
./gradlew runClient
```

In survival/creative, use a crafting table: place the three items in a single **horizontal** row in that order.
